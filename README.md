# BattleClass

Плагин для Minecraft (Paper 1.21+), добавляющий в игру военное снаряжение, оружие и системы для создания игрового процесса в стиле военного шутера.

## Описание

BattleClass - это комплексный плагин, обогащающий игровой процесс Minecraft военной тематикой. Он добавляет различные типы современного вооружения, бронежилеты, шлемы, медицинское оборудование, системы травм, военные звания и многое другое. Плагин предназначен для серверов, ориентированных на ролевые игры в военной тематике, PvP-серверов и серверов, где требуется реалистичное моделирование военных действий.

## Функциональность

### Оружие и боеприпасы
- Различные типы огнестрельного оружия (XM7, АК-12 и др.)
- Настройка характеристик оружия: урон, скорострельность, дальность, размер магазина
- Система аксессуаров для оружия
- Разные типы боеприпасов с индивидуальными характеристиками
- Отображение HUD патронов
- Огнеметы с системой топлива

### Защитное снаряжение
- Различные бронежилеты (6Б45, IOTV Gen 4)
- Военные шлемы (6Б47, ECH)
- Специальные ботинки и штаны с защитными свойствами
- Настраиваемые уровни защиты и прочности

### Медицинская система
- Медицинские предметы: бинты, аптечки, обезболивающее, адреналин
- Система травм: кровотечения, переломы
- Реалистичное лечение и восстановление
- Различные статус-эффекты от лекарств

### Инженерное оборудование
- Мины с детонацией и системой срока годности
- Минометы для дальнего боя
- Боевые дроны с аккумуляторами
- Взрывные устройства и гранаты

### Коммуникации
- Система раций с настраиваемыми частотами
- Военные планшеты для тактической информации

### Социальная система
- Военные звания с разными привилегиями
- Донатные возможности через встроенный GUI

## Установка

1. Скачайте последнюю версию плагина
2. Поместите JAR-файл в папку plugins на вашем сервере
3. Перезапустите сервер или используйте плагин для загрузки плагинов на работающем сервере
4. Настройте плагин в файле config.yml

## Требования

- Paper или Spigot сервер Minecraft версии 1.21 или выше
- Java 21 или выше

## Команды

### Основные команды
- `/battleclass` или `/bc` - основная команда для управления плагином
  - `/bc weapon <тип> [игрок]` - выдать оружие
  - `/bc attachments <оружие> <тип> [игрок]` - выдать аксессуары для оружия
  - `/bc list` - показать список доступного снаряжения
  - `/bc ammo <тип> <количество> [игрок]` - выдать патроны
  - `/bc select <тип>` - выбрать тип патронов
  - `/bc armor <тип> [игрок]` - выдать бронежилет
  - `/bc helmet <тип> [игрок]` - выдать шлем
  - `/bc boots <тип> [игрок]` - выдать ботинки
  - `/bc pants <тип> [игрок]` - выдать штаны
  - `/bc medical <тип> <количество> [игрок]` - выдать медицинские предметы
  - `/bc heal [игрок]` - вылечить игрока
  - `/bc injury [add|remove] <тип> [игрок]` - управление травмами
  - `/bc tablet [игрок]` - выдать планшет

### Команды для управления рангами
- `/rank` или `/ranks` или `/military` - команда для управления военными рангами
  - `/rank set <игрок> <ранг>` - установить ранг игроку
  - `/rank info [игрок]` - посмотреть информацию о ранге игрока
  - `/rank list` - показать список доступных рангов
  - `/rank remove <игрок>` - удалить ранг игрока

### Прочие команды
- `/donate` - открыть меню донатов
- `/r <сообщение>` - отправить сообщение через рацию
- `/r set <частота>` - установить частоту рации
- `/r off` - выключить рацию
- `/r info` - информация о текущей рации
- `/r help` - справка по использованию рации
- `/mortar [give|list|help]` - команда для управления минометами
- `/flamethrower [give|list|fuel] [игрок] [тип] [количество]` - команда для управления огнеметами
- `/drone [give|battery] [игрок] [тип|заряд]` - команда для управления боевыми дронами

## Разрешения

Плагин имеет обширную систему разрешений:

- `battleclass.admin` - полный доступ ко всем функциям плагина
- `battleclass.weapon` - доступ к получению оружия
- `battleclass.attachments` - доступ к получению аксессуаров для оружия
- `battleclass.list` - доступ к просмотру списка снаряжения
- `battleclass.ammo` - доступ к получению патронов
- `battleclass.select` - доступ к выбору типа патронов
- `battleclass.armor` - доступ к получению бронежилетов
- `battleclass.helmet` - доступ к получению шлемов
- `battleclass.boots` - доступ к получению ботинок
- `battleclass.pants` - доступ к получению штанов
- `battleclass.medical` - доступ к получению медицинских предметов
- `battleclass.heal` - доступ к исцелению игроков
- `battleclass.injury` - доступ к управлению травмами
- `battleclass.tablet` - доступ к получению военных планшетов
- `battleclass.rank.set` - доступ к установке военных рангов
- `battleclass.rank.remove` - доступ к удалению военных рангов
- `battleclass.rank.info` - доступ к просмотру информации о военных рангах
- `battleclass.donate` - доступ к открытию меню донатов
- `battleclass.radio` - доступ к получению радиостанций
- `battleclass.radio.command` - доступ к использованию команды рации
- `battleclass.mortar` - доступ к просмотру списка минометов
- `battleclass.mortar.give` - доступ к выдаче минометов
- `battleclass.flamethrower` - доступ к просмотру списка огнеметов
- `battleclass.flamethrower.give` - доступ к выдаче огнеметов
- `battleclass.flamethrower.fuel` - доступ к выдаче топлива для огнеметов
- `battleclass.drone` - доступ к просмотру помощи по дронам
- `battleclass.drone.give` - доступ к выдаче дронов
- `battleclass.drone.battery` - доступ к выдаче батарей для дронов

## Конфигурация

Основные настройки плагина находятся в файле `config.yml`. Там можно настроить:

- Общие параметры (дружественный огонь, множитель урона, дальность оружия)
- Параметры каждого вида оружия (урон, скорострельность, размер магазина и пр.)
- Характеристики защитного снаряжения
- Параметры медицинских предметов
- Настройки системы травм
- Множество других параметров для каждой функции плагина

## Сборка из исходного кода

Для сборки плагина из исходного кода:

1. Клонируйте репозиторий
2. Установите Java Development Kit 21 или выше
3. Используйте Gradle для сборки:
   ```
   ./gradlew build
   ```
4. Найдите собранный JAR-файл в директории `build/libs/`
